Sobre o Projeto

Este sistema foi desenvolvido para gerenciar funcionários e empresas de forma simples. Ele permite realizar todas as operações essenciais, como criar, editar, listar e excluir registros — tudo seguindo os princípios da arquitetura RESTful.

Tecnologias Utilizadas

O frontend foi construído utilizando a versão mais recente do Angular, aproveitando componentes standalone. A estilização segue um layout organizado com SCSS.

O backend é alimentado por uma API REST, responsável por toda a lógica de negócio e operações com o banco de dados. A comunicação entre o frontend e a API acontece através de requisições HTTP, garantindo uma arquitetura modular e robusta.

O que é uma API REST?

Uma API REST (Representational State Transfer) é um estilo arquitetural que organiza a comunicação entre sistemas utilizando os verbos HTTP, como GET, POST, PUT e DELETE. Cada requisição representa uma ação sobre um recurso, tornando o sistema previsível, escalável e fácil de manter.
